# 주요 기능

>- 걸음수 센서를 활용한 카운트 다운
>- acceleometer 센서를 활용한 벌레 먹이기 게임
>- LocationManager와 geoCoder를 활용한 현재 위치 및 주소 가져오기 
>- 네이버 지도 api를 활용한 벌레 잡은 위치 표시 및 클릭시 관련 정보 출력
